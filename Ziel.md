Webapp zur Erfassung von Stundenrapporten 
Ziel:
- kleine lokale Webapp basierend auf Docker mit kleiner Datenbank
- Erfassung von Daten einer Person mit Name, email, tel, adresse, der HW IT Infrastruktur, zb MacOS, Linux, Windows, Drucker
- Erfassung von login Daten Username PW für jedes Gerät, admin PW für Router, und sonstige Infrastruktur
- Hauptscreen: erfassung von Datum, Zeitdauer, Thema eines Support-Ereignisses bei einem Kunden (dh, einer erfassten Person) sowie Kosten/bzw erhaltenes Geld. Man müsste erfassen können, ob bar bezahlt wurde oder auf Rechnung.
